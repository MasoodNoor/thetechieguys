# The Techie Guys

Website for **The Techie Guys** — email and domain infrastructure specialists serving UK and international small businesses.

🌐 [thetechieguys.com](https://thetechieguys.com)

---

## Tech Stack

- **Framework** — Next.js 16
- **Styling** — Tailwind CSS v4
- **Components** — shadcn/ui
- **Deployment** — Vercel

---

## Local Development

```bash
pnpm install
pnpm dev
```
